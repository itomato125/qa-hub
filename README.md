# QA Hub — Test Management System

A modern, single-page web application for QA test management. No backend required — runs entirely in the browser.

## 🔗 Live Demo

👉 [https://YOUR-USERNAME.github.io/YOUR-REPO-NAME](https://YOUR-USERNAME.github.io/YOUR-REPO-NAME)

> Replace `YOUR-USERNAME` and `YOUR-REPO-NAME` after deploying.

## ✨ Features

- 📋 Test Case management (create, edit, delete, filter)
- 🐛 Defect tracking
- 📊 Dashboard with statistics
- 👥 Team management with roles (Admin / Function / Edit / View)
- 🔐 User login & session
- 📝 Activity Log & Login Log
- 🌙 Dark theme UI

## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository (public)
2. Upload `index.html` to the repo
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch`
5. Choose branch `main` and folder `/ (root)`
6. Click **Save**
7. Your site will be live at `https://YOUR-USERNAME.github.io/REPO-NAME`

## 🛠 Tech Stack

- React 18 (via CDN, no build step needed)
- Pure HTML/CSS/JS — single file
- Data stored in browser `localStorage`

## 📌 Notes

- All data is stored **locally in the browser** — no server or database needed
- Works offline after first load
- To reset all data: open browser DevTools → Application → Local Storage → Clear
